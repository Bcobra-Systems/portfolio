### 🏟️ Event Merchandise Logistics System

**Client:** Legends Global Merchandise | **Platform:** Power Apps + Power Automate + SharePoint

**The Problem:**
For major sporting events like the Super Bowl, merchandise teams across multiple locations needed to request specific inventory for their venues. The warehouse team had no structured way to receive pull requests, track packing progress, or document what was shipped. Coordination happened over email and phone calls, leading to missed items and no proof of what was packed.

**What We Built:**
An integrated two-app system: an inventory pull request app where event teams specify what merchandise they need by venue, and a shipping request app used by the warehouse team. Warehouse staff photograph every box packed and attach images to each order, creating a complete visual record. The system routes requests through approval, tracks fulfillment status, and provides end-to-end documentation from request to delivery.

**The Impact:**
- Streamlined merchandise logistics for high-profile national events
- Photo documentation of every packed shipment eliminated disputes
- Warehouse team gained a structured workflow replacing ad-hoc email coordination
- Full audit trail from pull request through packing to shipment

**Tech Stack:** `Power Apps` `Power Automate` `SharePoint` `Photo Capture` `Multi-App Integration`
